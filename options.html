🔗 API de IA con visión

⚡ La extensión siempre captura la pantalla y la envía a la IA. Necesitas un modelo con visión.

🔹 GitHub Models (gratis): gpt-4o-mini
  Endpoint: https://models.inference.ai.azure.com/chat/completions
  API Key: token de GitHub (ghp_...) en github.com/marketplace/models

🔹 Gemini (gratis): gemini-2.0-flash
  API Key: aistudio.google.com/apikey

🔹 OpenAI: gpt-4o-mini, endpoint https://api.openai.com/v1/chat/completions
Para Gemini se ignora (usa API nativa). Para OpenAI/GitHub Models sí se usa.
La clave del proveedor que elijas
Ej: gpt-4o-mini, gemini-2.0-flash, claude-3-5-sonnet, llama3.2-vision
